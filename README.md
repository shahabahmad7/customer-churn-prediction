# Customer Churn Prediction

This repository contains a Jupyter notebook for predicting customer churn using the Telco dataset.

## What is included

- `Notebook/customer_churn.ipynb`: walkthrough of data cleaning, exploratory analysis, and model training.
- `WA_Fn-UseC_-Telco-Customer-Churn.csv`: raw dataset used for the analysis.
- `Requirements.txt`: Python library dependencies.

## Goal

The notebook aims to identify which customers are most likely to leave, using several classification models and standard evaluation metrics.

## Highlights

- Data cleaning for duplicates, missing values, and data type corrections.
- Feature encoding and scaling for model readiness.
- Model comparisons: Logistic Regression, Decision Tree, Random Forest, SVM, XGBoost, CatBoost, and more.
- Accuracy, precision, recall, F1 score, confusion matrix, and ROC curves.

## How to run

1. Create a virtual environment and install dependencies from `Requirements.txt`.
2. Open `Notebook/customer_churn.ipynb` in Jupyter.
3. Run cells from top to bottom.
